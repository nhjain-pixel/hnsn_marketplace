# HNSN Next.js Portal Starter

## Quick start

1. Install Node.js on your computer
2. Open terminal in this folder
3. Run:

```bash
npm install
npm run dev
```

4. Open:
`http://localhost:3000`

## Deploy to Vercel

1. Upload this folder to GitHub
2. In Vercel, click **New Project**
3. Import the GitHub repo
4. Deploy
5. In Vercel project settings, add your GoDaddy domain
